## 当前产品边界

- 当前是 Vue 3 + Django 单体应用，并由统一 Scheduler 执行周期任务。
- AIOps 中的 Action、Skill、MCP、Runbook、巡检和告警研判是平台能力，不是独立部署的多 Agent 服务。
- 当前不提供 APM、Trace、服务调用链、无人值守自动修复或 PDF 巡检报告。
- Docker/容器环境代码属于遗留内部能力，当前产品菜单只提供 K8S 集群管理。
