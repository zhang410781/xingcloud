## 能力状态约定

- **当前已实现**：存在用户入口且代码链路完整。
- **有限支持**：能力存在，但仅在特定入口、数据源或操作范围内开放。
- **内部能力**：后端存在接口或兼容代码，前台未作为产品功能开放。
- **当前未实现**：不应在售前、用户指南或交付说明中承诺。
