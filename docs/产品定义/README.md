## 7 个智能体的模块依赖关系图

```
                        告警智能体
                         ├─ 告警引擎（规则评估+降噪）
                         ├─ 通知分发（企微/飞书/邮件/Webhook）
                         └─ 触发 → 故障研判智能体

                        故障研判智能体
                         ├─ AIOps 引擎（LLM + MCP）
                         ├─ 指标证据包（可观测性）
                         ├─ 日志分析智能体（可观测性）
                         ├─ K8S 事件（K8S 管理）
                         ├─ CMDB 资产关联（CMDB）
                         ├─ 知识图谱（业务上下文）
                         └─ 触发 → Runbook 智能体

                        日志分析智能体
                         ├─ AIOps 引擎（自然语言→查询参数）
                         └─ 日志数据源（Loki/ELK/ClickHouse）

                        巡检智能体
                         ├─ AIOps 引擎（报告生成）
                         ├─ 任务中心（巡检脚本执行）
                         ├─ 可观测性（健康指标）
                         ├─ CMDB（巡检范围）
                         └─ 通知（报告推送）

                        Runbook 智能体
                         ├─ Runbook 引擎（匹配+推荐+版本）
                         ├─ 任务中心（剧本步骤执行）
                         ├─ 故障研判（触发入口）
                         └─ 通知（随告警推荐）

                        异常检测智能体
                         ├─ 可观测性-指标源（数据来源）
                         ├─ 异常检测算法（3-Sigma/EWMA/IQR/IF）
                         ├─ 告警引擎（生成异常事件）
                         └─ 故障研判（可选分析）

                        变更智能体
                         ├─ 工单系统（变更触发入口）
                         ├─ CMDB（影响范围推导）
                         ├─ 可观测性（指标对比+实时监控）
                         ├─ 告警引擎（变更中异常检测）
                         └─ 通知（推送风险评估和对比报告）
```
