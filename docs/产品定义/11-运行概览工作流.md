# 运行概览（SLA 驾驶舱）— 产品工作流

## 1. 产品目标

运行概览是 SRE 数字员工的"仪表盘"，面向运维管理者和 SRE 组长。它聚合所有模块的数据，展示平台整体健康度、SLA 达标情况、告警趋势和工单及时率。

## 2. 展示内容

| 区域 | 数据来源 | 说明 |
|------|----------|------|
| SLA 状态 | 告警历史 | 本月 SLA（按灾难级告警持续时间计算） |
| 年度 SLA 目标 | SLA 历史 | 99.96% 目标是否可达成 |
| 产品 SLA | 告警历史 | 各业务分组/产品的 SLA |
| 工单统计 | 工单系统 | 本月工单数、处理及时率、超时、失败 |
| 告警统计 | 告警事件 | 告警数、级别分布、未确认数、风险项 |
| 资产概览 | CMDB | 资产总数、异常资产数 |

## 3. 工作流

```
用户在登录后默认进入运行概览
   → 查看本月 SLA 状态
   → 下钻：点击某产品 SLA → 查看该产品的告警事件明细
   → 查看告警趋势 → 可调整时间范围
   → 查看工单及时率 → 发现处理超时 → 跳转到工单列表跟进
```

## 4. 模块间依赖

| 输入来源 | 输出去向 |
|----------|----------|
| 告警事件 | SLA 计算 |
| 工单系统 | 工单统计 |
| CMDB | 资产概览 |
| 各模块异常数据 | 风险项列表 |
