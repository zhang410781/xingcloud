# 运行概览

> 状态：当前已实现

## 1. 产品定位

运行概览是平台整体驾驶舱，与可观测性的监控看板不同：

- 运行概览关注整个运维平台的资产、告警、任务、工单和风险态势。
- 监控看板关注当前业务上下文中的 K8S、服务器、数据库、中间件和日志指标。

## 2. 主要内容

当前后端汇总能力包括：

- 资产与 CMDB 概况。
- 告警数量、级别、状态和风险项。
- 任务与计划执行状态。
- 应用发布、事务工单和待审批事项。
- SLA/风险摘要等管理视角数据。
- 当前业务上下文相关信息。

前端实际展示内容受用户权限、模块开关和是否存在真实数据影响。

## 3. 业务上下文选择器

页面顶部的业务上下文选择器用于给多个模块提供统一默认范围。具体页面仍可能显示上下文名称或提供局部筛选，但不应使用与顶部选择器冲突的数据范围。

切换上下文后，可观测、告警、日志、K8S、资产和智能助手应读取对应上下文资源。

## 4. 空态

新交付环境没有资产、数据源、告警或工单时显示真实空态，不生成示例资产或虚假统计。

## 5. 当前边界

- 运行概览不是用户自定义 BI 大屏。
- SLA 数据属于平台已有汇总能力，不代表已经实现完整的 SLO 管理产品。
- 概览不替代各专业页面的详细查询和证据。
