# 工单与应用发布

> 状态：当前已实现；不存在独立“变更智能体”

## 1. 当前入口

- `工单系统 / 应用发布`
- `工单系统 / 事务工单`
- `工单系统 / 审批流`

## 2. 应用发布

应用发布面向 K8S 发布流程，支持发布记录、审批、分批推进、开始、停止、重跑、日志、状态详情和基于历史成功版本的回滚。

发布过程记录环境、集群、工作负载、镜像/版本、策略、审批和执行事件，并可同步相关 CMDB 关系。

## 3. 审批流

管理员可以配置发布审批步骤。发布在满足审批条件后进入执行，拒绝则终止当前流程。操作权限由发布查看、管理和审批权限控制。

## 4. 事务工单

事务工单支持创建、编辑、审批、拒绝、开始处理和完成，用于承接非发布类运维事项。

## 5. 变更证据

告警研判和巡检会查询同一业务上下文及时间窗口内的发布、任务和工单记录，将其作为候选变更证据。时间接近不等于因果关系，报告必须标记为待验证关联。

## 6. 当前边界

- 当前没有独立变更智能体、自动风险决策服务或无人值守回滚。
- 自动研判只能建议回滚评估或生成待确认任务，不能直接执行回滚。
- Docker 发布不属于当前新建发布流程。
